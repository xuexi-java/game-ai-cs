# 项目结构说明

## 当前项目结构

```
game-ai-cs/
├── backend/              # 后端服务 (Nest.js)
│   ├── src/
│   │   ├── app.module.ts
│   │   ├── main.ts
│   │   └── ...
│   ├── package.json
│   └── ...
├── player-app/          # 玩家端前端 (React + Vite)
│   ├── src/
│   ├── package.json
│   └── vite.config.ts
├── admin-portal/        # 管理端前端 (React + Vite)
│   ├── src/
│   ├── package.json
│   └── vite.config.ts
├── prisma/             # 数据库Schema
├── docs/               # 文档
└── package.json        # 根项目配置
```

## 下一步操作

1. 安装所有依赖：
   ```bash
   npm run install:all
   ```

2. 或者分别安装：
   ```bash
   cd backend && npm install
   cd ../player-app && npm install
   cd ../admin-portal && npm install
   ```

3. 启动开发服务：
   ```bash
   # 后端
   npm run dev:backend
   
   # 玩家端
   npm run dev:player
   
   # 管理端
   npm run dev:admin
   ```
