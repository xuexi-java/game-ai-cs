  配置说明                                                                                                              
                                                                                                                        
  模拟游戏服务器默认配置（mock-game-server.js 顶部）：                                                                  
                                                                                                                        
  const CONFIG = {                                                                                                      
    GAME_ID: 'test_game',      // 游戏ID（需在客服后台创建）                                                            
    SECRET: 'mysecretkey123',  // 签名密钥（与后台配置一致）                                                            
    NONCE: 'abc123nonce',      // 固定Nonce（与后台配置一致）                                                           
    PORT: 3001,                // 游戏服务器端口                                                                        
  };                                                                                                                    
                                                                                                                        
  重要： 需要在客服管理后台创建对应的游戏配置：                                                                         
  1. 访问 http://localhost:20101 登录管理后台                                                                           
  2. 进入 游戏管理 → 新建游戏                                                                                           
  3. 游戏名称填: test_game                                                                                              
  4. 启用玩家API: ✅                                                                                                    
  5. 签名密钥: mysecretkey123                                                                                           
  6. 固定Nonce: abc123nonce
